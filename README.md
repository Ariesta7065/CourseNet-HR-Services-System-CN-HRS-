# CourseNet-HR-Services-System-CN-HRS-
CourseNet HR Services System (CN-HRS) adalah aplikasi berbasis web yang dirancang untuk membantu manajemen SDM (Sumber Daya Manusia) di lingkungan Course Net Indonesia, sebuah perusahaan startup di bidang pelatihan dan edukasi teknologi.
Sistem ini bertujuan untuk mengotomatisasi proses HR seperti manajemen karyawan, absensi, rekrutmen, evaluasi kinerja, dan penggajian dalam satu platform yang efisien dan mudah digunakan.

Dengan menggunakan arsitektur berbasis container (Docker), sistem dapat dengan mudah di-deploy di berbagai lingkungan (development, staging, dan production) tanpa konflik dependensi.
